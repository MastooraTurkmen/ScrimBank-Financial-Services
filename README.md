# ScrimBank Financial Services


Quick start:

````
$ npm install
$ npm run dev
$ npm start
````

````
$ pnpm install
$ pnpm run dev
$ pnpm start
````


````
$ yarn
$ yarn run dev
$ yarn start
````
Head over to https://vitejs.dev/ to learn more about using vite
